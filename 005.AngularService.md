# âœ… **What is an Angular Service?**

**An Angular Service is a class used to share logic, data, or functionality across multiple components.**
It helps keep components clean and follow **separation of concerns**.

Typical uses:

* API calls (HTTP services)
* Shared state (cart, user session)
* Utility functions
* Business logic
* Inter-component communication

Angular services are usually **singleton** when provided in root.

# âœ… **1. How to Provide a Service**

### **A) Provide in Root (Most Common)**

```ts
@Injectable({ providedIn: 'root' })
export class UserService {}
```

âž¡ Automatically available everywhere (singleton).

---

### **B) Provide in a Component**

```ts
@Component({
  ...
  providers: [UserService]
})
export class HomeComponent {}
```

âž¡ Creates a **new instance** only for that component and its children.

---

### **C) Provide in a Module (old style)**

```ts

  providers: [UserService]
})
export class AppModule {}
```

âž¡ Not needed much after Angular 6+.

# âœ… **2. How to Consume (Use) a Service**

### **A) Inject in Constructor (Most Common)**

```ts
constructor(private userService: UserService) {}

ngOnInit() {
  this.userService.getUsers().subscribe(console.log);
}
```

---

### **B) Inject Using `inject()` (Angular 14+ / Standalone)**

```ts
const userService = inject(UserService);

userService.getUsers().subscribe(console.log);
```

Used inside:

* components
* services
* functions
* signals
* effects

---

### **C) Use Service Method in Template (through component)**

```html
<div>{{ userService.userName }}</div>
```

In component:

```ts
export class AppComponent {
  userService = inject(UserService);
}
```

---

### **D) Using a Service With Signals (Angular 16+)**

```ts
@Injectable({ providedIn: 'root' })
export class CounterService {
  count = signal(0);
  increment() { this.count.update(v => v + 1); }
}
```

Component:

```ts
counterService = inject(CounterService);
```

Template:

```html
<p>{{ counterService.count() }}</p>
<button (click)="counterService.increment()">+</button>
```

---

### **E) Using Service With RxJS Observables**

```ts
@Injectable({ providedIn: 'root' })
export class CartService {
  private items$ = new BehaviorSubject<any[]>([]);
  cartItems$ = this.items$.asObservable();

  addItem(item: any) {
    this.items$.next([...this.items$.value, item]);
  }
}
```

Component:

```ts
this.cartService.cartItems$.subscribe(items => console.log(items));
```

OR template:

```html
<div *ngFor="let item of cartService.cartItems$ | async">
    {{ item }}
</div>
```
