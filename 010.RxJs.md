# âœ… **What is RxJS?**

**RxJS (Reactive Extensions for JavaScript)** is a library for **reactive programming** using **Observables**.

* Allows you to work with **asynchronous streams of data**.
* Helps manage **events, HTTP requests, user input, timers, WebSockets** in a unified way.
* Integral part of Angular â€” `HttpClient`, `Forms`, `Router`, `Signals`, and NgRx all rely on RxJS.

**Key Concepts:**

1. **Observable** â€“ a stream of data (like a sequence of values over time)
2. **Observer** â€“ object that reacts to observable values (next, error, complete)
3. **Subscription** â€“ listening to an observable
4. **Operators** â€“ functions to transform, filter, or combine observables

| Operator Type                 | Examples                                                   | Use                                  |
| ----------------------------- | ---------------------------------------------------------- | ------------------------------------ |
| **Creation**                  | `of()`, `from()`, `interval()`, `timer()`                  | Create observables                   |

| **Utility / Side-effects**    | `tap()`, `finalize()`                                      | Debug, logging, or run side-effects  |
